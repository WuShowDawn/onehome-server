# onehome-server

Django+Vue实现移动端二手交易应用后端部分

前端部分：[onehome](https://github.com/michwh/onehome)

## 技术栈

* django
* django-rest-framework

## 实现功能

* [x] 登录
* [x] 注册
* [x] 发布信息
* [x] 获取商品列表

## api

|api                       |说明          |
|--------------------------|-------------|
|/users/login              |登录          |
|/users/register           |注册          |
|/product/getList/         |获取商品列表    |
|/product/getImagesList/   |获取商品图片列表 |
|/product/publish/         |发布商品消息    |
|/product/publishImages/   |发布商品图片    |
